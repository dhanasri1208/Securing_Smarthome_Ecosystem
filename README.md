# Securing a Smart Home Ecosystem

## Project Description
This project demonstrates the design and implementation of a Smart Home Security Monitoring System using Ubuntu Server as a centralized security hub. The system applies layered cybersecurity principles to protect smart home environments from common network-based attacks.

## Objectives
- Implement system hardening on Ubuntu Server
- Configure firewall rules using UFW
- Secure remote access using SSH hardening
- Deploy Snort Intrusion Detection System (IDS)
- Simulate attacks using Nmap and validate security controls

## Technologies Used
- Ubuntu Server 24.04 LTS
- UFW Firewall
- OpenSSH
- Snort IDS
- Nmap
- Oracle VirtualBox

## Key Features
- Least privilege enforcement
- Secure SSH configuration (root login disabled)
- Firewall-based traffic filtering
- Real-time intrusion detection
- Attack simulation and log analysis

## Project Structure
- **Report/** – Final project report (PDF)
- **Screenshots/** – Implementation and testing evidence
- **Config_Files/** – Sample configuration files
- **Demo/** – Video walkthrough link

## Outcome
The system successfully detects suspicious activities such as port scanning and unauthorized access attempts while restricting unnecessary services. This project proves that layered security significantly improves smart home network protection.
